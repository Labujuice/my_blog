Pyinstaller安裝與使用

Installationpip or pip3: pip3 install pyinstaller阿如果沒有PIP這邊複習一下pip安裝方法: apt -> sudo apt install python3-pip , pip下來之後可能還需要更新-> pip3 install --upgrade pip 完整打包指令 pyinstaller -F -w -c --clean target.py如果想要加上圖片則加上 --icon=thisisicon.ico注意事項沒有安裝完所有所需的套件一樣可以被打包成執行檔, 但是執行起來馬上會當掉完整打包會變得很肥, 如果說目標裝置已經有python以及其相關鎖需要的module, 則不一定需要完整打包
